
# PectJro

Project manager สำหรับบริหารจัดการโปรเจค


## Features

- ระบบสร้างบัญชี
- ระบบสร้างและกำหนดระยะเวลาโปรเจกต์
- ระบบเพิ่มสมาชิกเข้าไปในโปรเจกต์
- ระบบสร้างโน๊ตกำกับแต่ละโปรเจกต์
- ระบบแอดมิน
